About

This is a search engine for the New World Translation, Study Edition. It provides a Google like search experience that will help you to retrieve any scriptures in a blink of an eye which is what we need in both the internal and external ministry.

This search engine is powered by Typesense. The engine is typo tolarant and will figure out what you want based on the key words or phrase you input into the search bar. This powerful engine is also lighting fast, which gurantees a sub-50 milliseconds response time.

The scripture dataset is from JW.org which host the online version of the New World Translation of the Holy Scriptures, Study edition.
